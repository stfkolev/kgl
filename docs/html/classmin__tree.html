◆ check()

int min_tree::check ( graph &  g)
virtual

Checks whether algorithm can be applied.

The graph must

  • be undirected
  • be connected
  • have more than 2 nodes

Additionally the weights of the edges must have been set in advance using min_tree::set_distances.
